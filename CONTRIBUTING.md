# Contributing

Thanks for wanting to contribute! A short and simple guide:

- Report bugs and feature requests using Issues.
- Create a separate branch or fork and open a Pull Request against `main`.
  - Suggested branch names: `feature/your-feature` or `fix/short-description`.
- In the PR description, include what changed and how to test it.
- Build and test locally:
  - Open `DevLapse.sln` in Visual Studio 2022
  - Target: .NET 8.0
  - Verify the app starts and a short recording works
- Coding style: follow the existing C# style in the project. Keep commits small and focused.

Thank you!
